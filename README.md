## git commands
### Snapshot Commands
* git clone <url>             #clone a remote repository
* git status
* git add .                   #adds all files to git
* git rm
* git commit -m '<message>'   #commits changes from the index with a message

### Updating Commands
* git push                    #push committed changes to remote repository
